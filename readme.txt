Po wybraniu opcji start program przeprowadzi ankietę, która określi zbiór aut, którymi użytkownik jest zainteresowany.
Przy każdym pytaniu podane są wszystkie możliwe odpowiedzi.
W przypadku popełnienia błędu, można wrócić do menu głównego poprzez podanie opcji -1.
Po wyświetleniu przez algorytm propozycji, ankietę można uruchomić po raz kolejny.
